# How to use the Onefile version

- First, make sure you have downloaded it

- Put the file in the directory you want

- Then, double click the file and program will run




# How to use the Onefolder version

- First, make sure you have downloaded it

- Put the file in the directory you want

- Go to the YouTube downloader folder ----> dist

- Scroll down and look for YouTube Downloader.exe

- Double click and program will run
